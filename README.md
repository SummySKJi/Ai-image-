# AI Image Generator

Lightning-fast AI image generator with reference image support.

## Features
- Text-to-image generation
- Reference image uploads
- Image transformation
- Download results

## Tech Stack
- Next.js 14
- Stable Diffusion V3
- Tailwind CSS

## Getting Started
1. Clone the repo
2. Run `npm install`
3. Run `npm run dev`
4. Open http://localhost:3000
